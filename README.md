# hold-this-file

Serverless web app to temporarily store files for later redownloading across devices

Created for personal/family/small-scale use (using free-tier limitations on Supabase) and for learning about CRON jobs.

To-Be Implemented Features:

- Supabase auth
- Guest user support (by providing publicly accessible, unguessable download links)
- Cron job to delete files from Supabase DB on certain time intervals

Technologies used:

- React(Vite)
- Typescript
- React Router
- Supabase
- GitHub Actions or cron-job.org for cron
